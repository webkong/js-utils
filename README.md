# js-utils-live

JavaScript utils library.

[Utils List](./list.md)

## Install

```shell
npm install -S js-utils-live
```

## Usage

```javascript
const {date} =  require('js-utils-live')
const {format} = date;

or

import {date} from 'js-utils-live'
const {format} = date;
```
